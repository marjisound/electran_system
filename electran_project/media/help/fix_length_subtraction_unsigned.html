Addition and Subtraction in different Number Bases

The basic technique of adding and subtracting integers in decimal is extendible to addition of 2 integers in any base b.

It can be explained through the use of the power series representation in powers of b of integers, S and T:-

S = s0b0 + s1b1 + s2b2 + s3b3 + ............. + smbm + ...

T = t0b0 + t1b1 + t2b2 + t3b3 + ............. + tmbm + ...

Addition in base b

In the lectures we looked at the addition of S and T to give result R with power series representation:-

R = r0b0 + r1b1 + r2b2 + r3b3 + ............. + rmbm + ...

where all the coefficients r, s, and t are single symbols from the base b.

The value of r0, r1, r2, etc can be found by applying the following rule starting with r0:-

sm + tm + cm = rm + cm+1b

where c0, c1, c2 etc are carries between terms with c0=0.

The following examples are for decimal and binary:-

Addition in decimal: 735 + 896:-

cm+sm+ tm = rm+cm+1bm

Term 0: 0 + 5 + 6 = 11 : c1 = 1 r0 = 1

Term 1: 1 + 3 + 9 = 13 : c2 = 1 r1 = 3

Term 2: 1 + 7 + 8 = 16 : c3 = 1 r2 = 6

Term 3: 1 + 0 + 0 = 1 : c4 = 0 r3 = 1

Term 4: 0 + 0 + 0 = 0 : c5 = 0 r4 = 0

73510 + 89610 = 163110

Addition in binary: 111 + 011:-

cm+sm+ tm = rm+cm+1bm

Term 0: 0 + 1 + 1 = 10 : c1 = 1 r0 = 0

Term 1: 1 + 1 + 1 = 11 : c2 = 1 r1 = 1

Term 2: 1 + 1 + 0 = 10 : c3 = 1 r2 = 0

Term 3: 1 + 0 + 0 = 1 : c4 = 0 r3 = 1

Term 4: 0 + 0 + 0 = 0 : c5 = 0 r4 = 0

1112 + 0112 = 10102

but can be applied in any base:-

Addition in binary: 111 + 011:-

cm+sm+ tm = rm+cm+1bm

Term 0: 0 + 1 + 1 = 10 : c1 = 1 r0 = 0

Term 1: 1 + 1 + 1 = 11 : c2 = 1 r1 = 1

Term 2: 1 + 1 + 0 = 10 : c3 = 1 r2 = 0

Term 3: 1 + 0 + 0 = 1 : c4 = 0 r3 = 1

Term 4: 0 + 0 + 0 = 0 : c5 = 0 r4 = 0

1112 + 0112 = 10102

Subtraction in base b

The subtraction of T from S gives a result R with power series representation:-

R = r0b0 + r1b1 + r2b2 + r3b3 + ............. + rmbm + ...

where all the coefficients r, s, and t are single symbols from the base b.

The value of r0, r1, r2, etc can be found by applying the following rule starting with r0:-

sn - tn - wn = rn - wn+1b

where w0, w1, w2 etc are borrows between terms with w0=0.

The following examples are for decimal and binary:-

Subtraction in decimal: 163110 - 89610:-

sn - tn - wn = rn - wn+1b

Term 0: 1 - 6 - 0 = -5 = 5 - 10 : w1 = 1 r0 = 5

Term 1: 3 - 9 - 1 = -7 = 3 - 10 : w2 = 1 r1 = 3

Term 2: 6 - 8 - 1 = -3 = 7 - 10 : w3 = 1 r2 = 7

Term 3: 1 - 0 - 1 = 0 = 0 - 0 : w4 = 0 r3 = 0

Term 4: 0 - 0 - 0 = 10 = 0 - 0 : w5 = 0 r4 = 0

163110 - 89610 = 73510

Subtraction in decimal: 89610 - 163110:-

sn - tn - wn = rn - wn+1b

Term 0: 6 - 1 - 0 = 5 = 5 - 0 : w1 = 0 r0 = 5

Term 1: 9 - 3 - 0 = 6 = 6 - 0 : w2 = 0 r1 = 6

Term 2: 8 - 6 - 0 = 2 = 2 - 0 : w3 = 0 r2 = 2

Term 3: 0 - 1 - 0 = -1 = 9 - 10 : w4 = 1 r3 = 9

Term 4: 0 - 0 - 1 = -1 = 9 - 10 : w5 = 1 r4 = 9

Term 5 -> ? same as Term 4

89610 - 163110 = ..999...926510

Again the subtraction process can be done in any base:-

Subtraction in base 11: 65611 - 293811:-

sn - tn - wn = rn - wn+1b

Term 0: 6 - 8 - 0 = -2 = 9 - 1110 : w1 = 1 r0 = 9

Term 1: 5 - 3 - 1 = 1 = 1 - 0 : w2 = 0 r1 = 1

Term 2: 6 - 9 - 0 = -3 = 8 - 1110 : w3 = 1 r2 = 8

Term 3: 0 - 2 - 1 = -3 = 8 - 1110 : w4 = 1 r3 = 8

Term 4: 0 - 0 - 1 = -1 = 1010 - 1110 : w5 = 1 r4 = A11

Term 5 -> ? same as Term 4

65611 - 293811 = ..AAA...A881911

Alternatively,

subtraction can be done by taking the complement of the value to be subtracted and then performing addition:-

Subtraction in base 11: 65611 - 293811 = 65611 + .AAA817311 = .AAA881911 = A881911
