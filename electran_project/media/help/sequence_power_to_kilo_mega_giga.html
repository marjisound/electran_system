Help page for Kilo, Mega and Giga conversions

These questions are to help make you familiar with binary K, binary M and binary G.

The table at the end has powers of 2 and their K, M or G equivalents.

The questions are in 3 forms.

The first form is a question such as :-

Express the following in binary K,M, or G, e.g 15K, 23M, 507G :-

  • 2^11+2^12+2^19

This is obviously less than 2^20, i.e. 1 Meg, so our answer will be Ks.

First you need to divide each term by 1K (2^10) to the total in K, and then convert the result into decimal

  • (2^11+2^12+2^19)/2^10
  • = (2 + 2^2 + 2^9) K
  • = (2 + 4 + 512) K
  • = 518K

The second form has questions such as:-

Question Cw3_d: Below is the sum of a sequence of power terms, where each term is a different power from the range of i values shown in brackets.

Express the sum in binary K,M, G:-

  • Sum: 2^ (i=12----->i=16 )

First the sum has to be expanded, thus:-

  • 2^12 + 2^13 + 2^14 + 2^15 +2^16

and then asnwered as for the 1st form.

The 3rd form of question is like this:-

Question Cw3_f: Below is the sum of a sequence of power terms, where each term is a different power from the range of i values shown in brackets.

Express the sum in binary K,M, G:-

  • Sum: [2^ ( i=30----->i=38 )] + 2^30

This solvable in the same way as the second form, but there is a short cut that can be made, which you should try to identify. (Hint: try doing the addition before transforming into decimal.)

Powers of 2: their values, convenient representations and approximations:
20 = 1; 210 = 1,024 = 1 binary K; 220 = 1,048,576 = 1 binary M;
21 = 2; 211 = 2,048 = 2 binary K; 221 = 2,097,152 = 2 binary M;
22 = 4; 212 = 4,096 = 4 binary K; 222 = 4,194,304 = 4 binary M;
23 = 8; 213 = 8,192 = 8 binary K; 223 = 8,388,608 = 8 binary M;
24 = 16; 214 = 16,384 = 16 binary K; 224 = 16,777,216 = 16 binary M;
25 = 32; 215 = 32,768 = 32 binary K;    
26 = 64; 216 = 65,536 = 64 binary K; 230 = 1,073,741,824 = 1 binary G;
27 = 128; 217 = 131,072 = 128 binary K; 232 = 4,294,967,296 = 4 binary G;
28 = 256; 218 = 262,144 = 256 binary K;    
29 = 512; 219 = 524,288 = 512 binary K; 240 = 1,099,511,627,776 = 1 binary T;

Notes: 1 binary K ≃ 1000; 1 binary M ≃ 1000 binary K ≃ 1,000,000; 1 binary G ≃ 109;

1 binary T ≃ 1012

K ≡ Kilo or K ; M ≡ Mega or Meg; G ≡ Giga or Gig; T ≡ Tera

The binary is not usually pronounced so 216 is usually just "64 K", 232 just "4 Giga" or "4 Gig" etc.
