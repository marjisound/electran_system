Instruction Execution Question

This question asks you to examine an instruction, such as that in the diagram ?slt $12, $20, $23?, and to identify the changes made to registers and or memory by the instruction during the fetch and execution of the instruction.

The screen for this question is divided into 3 parts: the upper section displays the question and the answer form; the lower left displays the Mips processor registers and their contents (registers pc, $0 to $31); the lower right a section of the Mips memory (locations 0x10000000-0x1000001F and their contents in the example in the figure).
The last 2 provide the data for the question. All data values are hexadecimal.

You are expected to work out what the instruction does, discover the source data values by looking up the register and memory contents, determine the result value of the instruction from the source values, and identify where this value is placed.

You are required to enter into the form all the changes that are made to registers and memory during the fetch and execution of the instruction.

Thus if a register in the register file (registers $0 to $31) is changed, you need to enter that the register file is written, enter the number of the register written and also enter the new value of this register (in hexadecimal).

Note that all register changes during the fetch and execution of the instruction must be entered.
Remember that memory accesses can access 1, 2 or 4 bytes from memory depending on the instruction.
When more than one byte is accessed in memory, you need to remember that the memory is big-endian, i.e. the most significant bytes of the value read or written come from or go to the lower memory addresses.
